twistedActor

twistedActor is core code for writing an actor: an instrument control computer (ICC) or other controller that executes commands and outputs replies in APO-standard keyword=value format. twistedActor communicates using TCP/IP and supports multiple users. twistedActor is based on an older package named TclActor.

Documentation includes:

  • Standard Commands for the standard commands and replies provided by twistedActor.Actor.
  • Writing an Actor using twistedActor.
  • Source code documentation generated by Doxygen.
  • Version history
  • SDSS keyword dictionary for the standard keywords output by twistedActor. Add them to the SDSS keyword dictionary for any actor based on twistedActor.

Installation

twistedActor is written in pure python. It requires the following 3rd party python packages:

  • Twisted framework for event-based communications. As of 2014-08-29 I am using Twisted version 14.
  • RO package for a communication framework and other utilities. As of 2014-08-28 I am using version 3.5.3.

twistedActor may be installed using the standard distutils command: python setup.py install. It is also compatible with the eups package distribution system (hence the ups directory). If using eups then twistedActor need and only be declared or installed (not built) to be used.
