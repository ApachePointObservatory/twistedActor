TwistedActor

TwistedActor is core code for writing an actor: an instrument control computer (ICC) or hub-based script that does something useful.

See Standard Commands for the standard commands and replies provided by TwistedActor.Actor.

See TwistedActor Internals for information about writing code based on the TwistedActor package.
